## 安装使用

### 1. 安装主题

将`xirong`文件夹上传到EMLOG的`content/templates/`目录：

```
content/templates/xirong/
```

### 2. 启用插件

在EMLOG后台启用以下插件：
- **模板设置（tpl_options）** - 必需

### 3. 激活主题

在EMLOG后台 → 外观 → 模板，选择"息壤主题"并激活。

### 4. 配置主题

激活后，进入"模板设置"进行详细配置：

#### 基础设置
- LOGO显示方式（文字/图片）
- 导航链接配置

#### 首页设置
- 首屏标题和副标题
- CTA按钮配置
- 各区块标题自定义
- FAQ问答内容
- 关于我们/加入我们内容

#### 页脚设置
- Footer链接列配置（4列）
- 版权信息
- 法律文档内容

## 自定义开发

### 修改配色

编辑`css/style.css`，修改CSS变量：

```css
:root {
    --color-primary: #0066FF;        /* 主色 */
    --color-background: #FFFFFF;     /* 背景色 */
    --color-text-primary: #1A1A1A;  /* 文字颜色 */
}
```

### 添加新功能

在`module.php`中添加新的PHP函数，在`js/main.js`中添加JavaScript功能。

## 常见问题

### Q: 如何修改LOGO？

A: 后台 → 模板设置 → 基础设置 → LOGO显示方式，选择"图片"并上传。

### Q: 如何修改导航菜单？

A: 后台 → 模板设置 → 基础设置 → 导航链接，每行一个，格式：`导航名称|链接地址`

### Q: 如何修改首屏内容？

A: 后台 → 模板设置 → 首页设置，修改相应字段。

### Q: 深色模式不工作？

A: 检查浏览器控制台是否有JavaScript错误，确保`js/main.js`正确加载。
