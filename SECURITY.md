# 安全漏洞报告指南

感谢您的安全研究和负责任的漏洞披露！请不要在 GitHub Issue 中公开报告安全漏洞。

如需报告安全问题，请用以下方式提交：

- 通过 [GitHub Security Advisories](https://github.com/emlog/emlog/security/advisories) 提交

我们会尽快确认并回复您的报告，被确认且修复后的问题，为避免被恶意利用，3个月后我们会为其申请CVE并公开。

谢谢您的支持！

# Security Vulnerability Reporting Guidelines

Thank you for your security research and responsible vulnerability disclosure! Please do not publicly report security vulnerabilities in GitHub Issues.

To report a security issue, please submit it via:

- [GitHub Security Advisories](https://github.com/emlog/emlog/security/advisories)

We will acknowledge and reply to your report as soon as possible. For confirmed and fixed issues, to avoid malicious exploitation, we will apply for a CVE and disclose it after 3 months.

Thank you for your support!
