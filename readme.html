介绍

  • emlog 是 "Every Memory Log" 的简称，意即：点滴记忆。它是一个功能强大的个人博客(blog)系统。致力于为您提供高效、稳定，且在使用上又极其简单、舒适的博客服务。

安装说明

  • 1、将src文件夹下的所有文件上传到服务器预安装emlog的根目录。
  • 2、在浏览器上访问根目录下的安装文件 install.php 按照提示填写安装。
  • 3、安装成功，删除安装文件: install.php

功能说明

  • 一键式更换模板，方便快捷打造个性博客
  • 人性化的日志撰写页面、配以自动保存功能，书写博文更加舒适
  • 日志草稿箱功能，方便保存你未完成的日志
  • 灵活的侧边栏组件(widgets)管理，让你轻松组合、自定义自己喜欢的组件
  • 独有的twitter功能，让你用简单的文字记录生活
  • 首页日历方式查阅日志，方便、直观、快捷
  • 支持手机wap功能，随时随地记录你的生活
  • 文本缓存设计，博客访问速度更快
  • 整体使用utf-8编码方式，让你的博客和世界接轨。
  • 支持灵活的标签(tag)分类，以及传统分类方式
  • 后台使用跨浏览器可视化编辑日志，支持ie\firefox\safari等浏览器
  • 方便的附件（文件、图片）上传和管理
  • 上传的图片可以随意直观的嵌入到日志内容里，让你的日志图文并茂
  • 支持引用通告(trackback) , 并配有强大的垃圾引用防御功能
  • 支持RSS日志输出功能 ，方便朋友订阅关注你的博客
  • 数据库备份/恢复功能
  • 访问量统计

模板说明

  • emlog支持一键式更换模板，让你方便快捷的打造自己的个性博客。官方首页为您免费提供了许多漂亮的模板下载，如果你喜欢可以下载使用。 如何安装从官方下载的模板?
      1、把解压后的模板文件夹复制或上传到 content/templates 目录下。
      2、登录后台进入模板管理你将可以看到模板列表中已经有了你刚才添加的模板，点击使用它，OK！
